-------------------Teacher User Story-------------------
- As a teacher, I want to register an account, so that I can access platform. [6]
- As a teacher, I want to log in to my account, so that I can access my dashboard. [7]
- As a teacher, I want to create an exam, so that I can collect the essays from my students. [8]
- As a teacher, I want AI to automatically grade essay asignments, so that I can return results to students faster. [9]
- As a teacher, I want to create a class, so that I can easily manage my students. [10]
- As a teacher, I want to edit class information, so that I can keep my class details updated. [11]
- As a teacher, I want to delete a class, so that I can remove classes that no longer active. [12]
- As a teacher, I want to view all my classes, so that I can navigate between them. [13]
- As a teacher, I want to view class details, so that I can monitor class information and manage class activities. [14]

-------------------Teacher User Flow-------------------
Flow 01 - Register Account
- Feature: register an account
- Role: teacher
- Flow: 
   1. Open website
   2. Click “Sign up”
   3. Select teacher role
   4. Fill in registration form
        - Fullname
        - Email/Phone number
        - Password
        - Confirm password
        - Birthday
        - Schools (E.x: THPT Bui Thi Xuan, ….)
   5. Click “Register”
   6. System validates information
   7. Account created successfully
   8. Redirect to Teacher Dashboard
----------------------------------------------------------------
Flow 02 - Log in to account
- Feature: log in to account
- Role: teacher
- Flow:
   1. Open website
   2. Click “Login”
   3. Enter login information
        - Email/Phone number
        - Password
   4. Click “Login”
   5. System verifies credentials
   6. Redirect to Teacher Dashboard
----------------------------------------------------------------
Flow 03 - Create Exam
- Feature: create exam
- Role: teacher
- Flow:
   1. Login to system
   2. Go to classes
   3. Select a class
   4. Click “Create Exam”
   5. Fill in exam information
        - Exam title
       - Instructions
        - Essay questions (E.x: PDF, docx, latex, …)
   6. Click “Publish Exam”
   7. System creates exam
   8. Students can submit essays
----------------------------------------------------------------
Flow 04 - AI Grade Essay
- Feature: AI grading essay
- Role: teacher
- Flow:
   1. Login to system
   2. Go to classes
   3. Select a class
   4. Select an exam
   5. View list of student submissions
   6. Click “AI Grade”
   7. System processes essay using AI model
   8. AI generates
        - Score
        - Feedback
        - Suggestions
   9. Teacher reviews result
   10. Click “Publish Result”
   11. Students receive results
----------------------------------------------------------------
Flow 05 - Create class
- Feature: create class
- Role: teacher
- Flow:
   1. Login to system
   2. Access Dashboard
   3. Click “Classes”
   4. Click “Create Class”
   5. Fill in class information
        - Class name
        - Subject
        - Description
        - Academic year
   6. Click “Create”
   7. System saves class
   8. Redirect to Class List
----------------------------------------------------------------
Flow 06 - View all classes
- Feature: view all classes
- Role: teacher
- Flow:
   1. Login to system
   2. Access Dashboard
   3. Click “Classes”
   4. System displays list of classes
        - Class name
        - Subject
       - Number of students
       - Created date
  5. Teacher selects a class to view details
----------------------------------------------------------------
Flow 07 - Edit Class
- Feature: edit class information
- Role: teacher
- Flow:
  1. Login to system
  2. Go to classes
  3. Select a class
  4. Click “Edit class”
  5. Update class information
       - Class name
       - Subject
       - Description
  6. Click “Save changes”
  7. System updates class information
----------------------------------------------------------------
Flow 08 - Delete class
- Feature: delete class
- Role: teacher
- Flow:
   1. Login to system
   2. Go to classes
   3. Select a class
   4. Click “Delete Class”
   5. System shows confirmation dialog
   6. Click “Confirm Delete”
   7. System deletes class
   8. Redirect to class list
----------------------------------------------------------------
Flow 09 - View class detail
- Feature: view class detail
- Role: student
- Flow:
   1. Login to system
   2. Access Dashboard
   3. Click classes
   4. System displays list of classes
   5. Select a class
   6. System opens class detail page
   7. Teacher can view
        - Class information
        - List of exams
        - Student list
        - Submissions
