-------------------Student User Story-------------------
- As a student, I want to register an account, so that I can access platform. [1]
- As a student, I want to log in to my account, so that I can access my dashboard. [2]
- As a student, I want to submit my essay, so that it can be graded by the system. [3]
- As a student, I want to join a class using class code, so that I can access course materials and activities. [4]
- As a student, I want to view all joined classed, so that I can navigate between them. [5]
- As a student, I want to view class details, so that I can access assignments and course activities. [6]
-------------------Student User Flow-------------------
Flow 01 - Register Account
- Feature: register an account
- Role: student
- Flow: 
   1. Open website
   2. Click “Sign up”
   3. Select student role
   4. Fill in registration form
        - Fullname
        - Email/Phone number
        - Password
        - Confirm password
        - Birthday
        - Schools (E.x: THPT Bui Thi Xuan, ….)
   5. Click “Register”
   6. System validates information
   7. Account created successfully
   8. Redirect to Student Dashboard

-------------------Teacher User Story-------------------
- As a teacher, I want to register an account, so that I can access platform. [6]
- As a teacher, I want to log in to my account, so that I can access my dashboard. [7]
- As a teacher, I want to create an exam, so that I can collect the essays from my students. [8]
- As a teacher, I want AI to automatically grade essay asignments, so that I can return results to students faster. [9]
- As a teacher, I want to create a class, so that I can easily manage my students. [10]
- As a teacher, I want to edit class information, so that I can keep my class details updated. [11]
- As a teacher, I want to delete a class, so that I can remove classes that no longer active. [12]
- As a teacher, I want to view all my classes, so that I can navigate between them. [13]
- As a teacher, I want to view class details, so that I can monitor class information and manage class activities. [14]
